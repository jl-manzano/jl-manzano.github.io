Sobre mí

  • Trabajo en equipo
  • Bases de datos (SQL, MySQL, PostgreSQL)
  • Control de versiones con Git
  • Desarrollo de aplicaciones móviles (Android, iOS)
  • Desarrollo de APIs RESTful y consumo de APIs externas
  • Diseño de interfaces de usuario (UI/UX)

Experiencia:
